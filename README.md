# 热门技术栈学习资源清单

这份清单旨在为你提供一些当前热门技术栈的学习资源，你可以根据自己的兴趣和需求，将这些资源整理到你的 GitHub 学习仓库中。

## 1. 编程语言 (Programming Languages)

| 语言 | 推荐资源 | 描述 |
| :--- | :--- | :--- |
| **Python** | [FreeCodeCamp](https://www.freecodecamp.org/), [Coursera](https://www.coursera.org/courses?query=programming%20languages), [Codecademy](https://www.codecademy.com/catalog/language/python) | 广泛应用于数据科学、Web 开发、自动化等领域，易学易用。 |
| **JavaScript/TypeScript** | [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript), [Codecademy](https://www.codecademy.com/catalog/language/javascript), [Udemy](https://www.udemy.com/) | Web 前端和后端（Node.js）开发的核心语言，TypeScript 提供了类型安全。 |
| **Go** | [Go 官方文档](https://go.dev/doc/), [Udemy](https://www.udemy.com/) | 谷歌开发的编译型语言，以高性能和并发性著称，适用于后端服务和微服务。 |
| **Java** | [Coursera](https://www.coursera.org/courses?query=programming%20languages), [Udemy](https://www.udemy.com/) | 企业级应用开发的主流语言，拥有庞大的生态系统和社区支持。 |
| **Rust** | [Rust 官方文档](https://www.rust-lang.org/learn), [The Rust Programming Language Book](https://doc.rust-lang.org/book/) | 以内存安全和高性能为特点，适用于系统编程、WebAssembly 等。 |

## 2. Web 开发 (Web Development)

### 前端框架 (Frontend Frameworks)

| 框架 | 推荐资源 | 描述 |
| :--- | :--- | :--- |
| **React** | [React 官方文档](https://react.dev/learn), [Codecademy](https://www.codecademy.com/catalog/subject/web-development) | Facebook 维护的 JavaScript 库，用于构建用户界面，生态系统庞大。 |
| **Vue.js** | [Vue.js 官方文档](https://vuejs.org/guide/introduction.html), [Udemy](https://www.udemy.com/) | 渐进式 JavaScript 框架，易于上手，适用于构建单页应用。 |
| **Tailwind CSS** | [Tailwind CSS 官方文档](https://tailwindcss.com/docs) | 实用至上的 CSS 框架，通过组合类名快速构建自定义设计。 |

### 后端框架 (Backend Frameworks)

| 框架 | 推荐资源 | 描述 |
| :--- | :--- | :--- |
| **Node.js (Express)** | [Node.js 官方文档](https://nodejs.org/en/docs/), [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Learn_web_development/Node_basics) | 基于 Chrome V8 引擎的 JavaScript 运行时，Express 是其流行的 Web 框架。 |
| **Django (Python)** | [Django 官方文档](https://docs.djangoproject.com/en/stable/) | Python 的全栈 Web 框架，以“电池包含”为理念，快速开发复杂应用。 |
| **Flask (Python)** | [Flask 官方文档](https://flask.palletsprojects.com/en/latest/) | Python 的微型 Web 框架，轻量灵活，适合小型项目和 API 开发。 |

## 3. 云计算与 DevOps (Cloud Computing & DevOps)

| 领域 | 推荐资源 | 描述 |
| :--- | :--- | :--- |
| **AWS (Amazon Web Services)** | [AWS 官方文档](https://docs.aws.amazon.com/), [AWS Skill Builder](https://aws.amazon.com/training/digital-training/) | 全球领先的云计算平台，提供广泛的服务，包括计算、存储、数据库等。 |
| **Azure (Microsoft Azure)** | [Microsoft Learn](https://learn.microsoft.com/en-us/azure/) | 微软的云计算平台，与微软生态系统紧密集成，提供企业级解决方案。 |
| **Google Cloud (GCP)** | [Google Cloud 官方文档](https://cloud.google.com/docs), [Google Cloud Learning Paths](https://cloud.google.com/learn) | 谷歌的云计算平台，在数据分析、机器学习和 Kubernetes 方面表现出色。 |
| **Docker** | [Docker 官方文档](https://docs.docker.com/get-started/), [KodeKloud](https://kodekloud.com/courses/docker-for-the-absolute-beginner/) | 容器化技术，实现应用及其依赖的打包、分发和运行。 |
| **Kubernetes** | [Kubernetes 官方文档](https://kubernetes.io/docs/home/), [KodeKloud](https://kodekloud.com/courses/kubernetes-for-the-absolute-beginner/) | 容器编排平台，用于自动化部署、扩展和管理容器化应用。 |

## 4. 学习平台 (Learning Platforms)

*   **FreeCodeCamp** [1]：提供免费的编程课程和认证，涵盖 Web 开发、Python 等。
*   **Coursera** [2]：与大学和公司合作，提供高质量的在线课程和专业证书。
*   **Codecademy** [3]：交互式学习平台，通过实践项目学习编程语言和技术。
*   **Udemy** [4]：拥有大量付费和免费课程，涵盖各种技术主题。
*   **MDN Web Docs** [5]：Mozilla 维护的 Web 技术文档，是前端开发的权威参考。
*   **KodeKloud** [6]：专注于 DevOps 和云技术的实践学习平台。
